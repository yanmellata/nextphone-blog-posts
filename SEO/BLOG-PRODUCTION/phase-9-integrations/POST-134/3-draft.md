## How to Set Up NextPhone with Freshsales (5-Minute Webhook Guide)

Setting up the integration is surprisingly simple. No developer required.

### Prerequisites (What You Need)

- Active Freshsales account (any plan)
- NextPhone account
- Your Freshsales API key (found in Settings → API Settings)

### Webhook Configuration Steps

1. **Log into NextPhone dashboard**
2. **Navigate to Integrations → Custom Webhooks**
3. **Click "Add New Integration"** and select HTTP Webhook
4. **Configure the webhook:**
   - Tool name: "Send Lead to Freshsales"
   - HTTP method: POST
   - URL: `https://[yourinstance].freshsales.io/api/leads`
   - Headers: Add your Freshsales API key
5. **Map the fields:**
   - `first_name` → Caller's first name (collected by AI)
   - `phone` → Caller's phone number
   - `email` → Caller's email (if collected)
   - `description` → Call summary generated by AI
6. **Set trigger:** Configure when to send (e.g., after every call, or only when caller requests callback)

### Testing Your Integration

Make a test call to your NextPhone number. Provide sample information when the AI asks.

Then check your Freshsales dashboard. You should see a new lead appear with:

- Contact information
- Call notes
- Timestamp
- Source tagged as "NextPhone"

The entire setup takes about 5 minutes. You can be live the same day.
