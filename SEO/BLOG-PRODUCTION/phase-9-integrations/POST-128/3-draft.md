### The Property Qualification Script

Here's what happens when someone calls about your listing.

## NextPhone + LionDesk Integration Setup

### Three Ways to Connect

You have options depending on your technical comfort level.

**Option 1: HTTP Webhooks (Direct API)**

NextPhone sends call data directly to the LionDesk API endpoint. You configure the webhook URL (https://api-v2.liondesk.com/), add OAuth authentication, and map which fields go where (name, phone, notes, tags).

Setup time: 15-20 minutes.

**Option 2: Zapier Integration (No-Code)**

Connect NextPhone and LionDesk through Zapier. When a call ends, Zapier creates a new contact in LionDesk and adds call notes. No coding required.

Setup time: 5 minutes.

**Option 3: Manual Review + Entry**

NextPhone logs all call details. You receive an email notification after each call. Review the transcript and manually add leads to LionDesk.

This defeats the automation purpose, but it works if you want to test before committing to integration.

### What Data Syncs Automatically

When someone calls about your listing, here's what appears in LionDesk:

- A new contact is created (or existing contact updated)
- Call notes are attached: "Interested in 123 Elm St, pre-approved for $500K, wants showing this weekend"
- A task is created for follow-up: "Call back before 5 PM Friday"
- Tags are added: "Hot Lead," "Elm St Property," "Pre-Approved"

Think of it like email forwardingwhen NextPhone takes a call, it automatically creates a contact in your CRM with all the details. No manual entry.
